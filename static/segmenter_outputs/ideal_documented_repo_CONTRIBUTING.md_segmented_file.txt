- Report Bugs: Found a bug? Create an issue with details such as steps to reproduce, expected behavior, and any relevant screenshots or logs.

- Pull Requests: Review the [Pull Request Template](./PULL_REQUEST_TEMPLATE.md) for submitting your changes. Ensure your code is well-documented and adheres to our style guidelines.

- Follow the [Installation Guide](./INSTALLATION_GUIDE.md) to install dependencies and set up your environment.
